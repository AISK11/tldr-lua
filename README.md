# TL;DR - Lua

*- personal notes for the Lua programming language.*

## Installation

### Linux

#### Arch Linux

1. Install Lua interpreter:
    ```sh
    lua -v || pacman -S --needed --noconfirm lua
    ```

## Usage

1. Execute script:
    ```sh
    lua <script>.lua
    ```
